# Projeto Academia

Este é um projeto web desenvolvido em [Next.js](https://nextjs.org/) utilizando o App Router, configurado para proporcionar uma experiência moderna, rápida e responsiva.

## 🚀 Tecnologias Utilizadas

O projeto está estruturado com as seguintes tecnologias principais:

- **Next.js 16 (App Router):** Framework React para desenvolvimento de aplicações web modernas com renderização no lado do servidor (SSR) e geração de sites estáticos (SSG).
- **React 19:** Biblioteca principal para construção de interfaces declarativas e baseadas em componentes.
- **Tailwind CSS v4:** Framework de estilização baseado em classes utilitárias para desenvolvimento visual ágil e moderno.
- **ESLint:** Ferramenta para linting e garantia de padrões de qualidade de código.

---

## 🛠️ O que foi feito até o momento

- **Estruturação do Projeto:** Inicialização do ecossistema Next.js com a estrutura do App Router (`src/app/`).
- **Configuração de Estilos Globais:** Integração do Tailwind CSS v4 e configuração do arquivo de estilos globais (`src/app/globals.css`), incluindo suporte básico para temas claro (light) e escuro (dark).
- **Configuração de Fontes e Metadados:** Definição das fontes padrão (`Geist` e `Geist Mono`) e metadados base da aplicação no arquivo de layout raiz (`src/app/layout.js`).
- **Configuração de Qualidade de Código:** Setup do ESLint com regras recomendadas para Next.js.
- **Integração com Versionamento:** Configuração inicial do repositório Git e publicação do projeto no GitHub ([projeto_academia](https://github.com/gustaaxz/projeto_academia.git)).

## 💻 Como rodar o projeto localmente

Siga as instruções abaixo para executar o servidor de desenvolvimento em sua máquina local:

### 1. Instalar as dependências

No diretório do projeto, execute o gerenciador de pacotes de sua preferência para instalar as dependências:

```bash
npm install
# ou
yarn install
# ou
pnpm install
```

### 2. Iniciar o servidor de desenvolvimento

Após instalar as dependências, inicie o servidor:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado do projeto rodando em tempo real.
